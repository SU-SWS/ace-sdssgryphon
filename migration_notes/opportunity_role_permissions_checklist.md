# Opportunity Role Permissions Checklist

For each of the following roles, ensure the permissions below are assigned for the Opportunity content type and its referenced vocabularies:

## Roles:
- site_manager
- site_editor
- contributor

## Permissions to Assign:
- create stanford_opportunity content
- edit any stanford_opportunity content
- edit own stanford_opportunity content
- delete any stanford_opportunity content
- delete own stanford_opportunity content
- view any unpublished stanford_opportunity content
- view stanford_opportunity revisions
- revert stanford_opportunity revisions
- delete stanford_opportunity revisions
- create terms in opportunity_sponsor
- edit terms in opportunity_sponsor
- delete terms in opportunity_sponsor
- create terms in opportunity_type
- edit terms in opportunity_type
- delete terms in opportunity_type
- create terms in opportunity_tag_filters
- edit terms in opportunity_tag_filters
- delete terms in opportunity_tag_filters

---

_Review and update these permissions in the relevant role config files or via the UI after import._
